#Console Commands

## Generic
    vr.HMDVersion                                   Prints Oculus SDK version used and Oculus Plugin info
    vr.bEnableHMD [true|false]                      Enables/Disables HMD
    vr.bEnableStereo [true|false]                   Enables/Disables Stereo mode
    vr.WorldToMetersScale [<float>]                 Changes worldunits-to-meters scale
    vr.HiddenAreaMask [true|false]                  Enables/Disables hidden-area mask
    vr.MirrorMode [<int>]                           Sets mirror mode
        0=Disabled, 1=SingleEye Letterboxed, 2=Undistorted, 3=Distorted, 4=SingleEye, 5=SingleEye Cropped
        
    vr.TrackingOrigin [floor|eye]                   Sets tracking origin
    vr.HeadTracking.Reset <yaw>                     Resets position and orientation, applies yaw (in degrees)
    vr.HeadTracking.ResetPosition                   Resets position only
    vr.HeadTracking.ResetOrientation <yaw>          Resets orientation only, applies yaw (in degrees)
    vr.HeadTracking.Status                          Shows current head tracking status
    vr.oculus.bUpdateOnRenderThred [<bool>]         Enable/Disable updating HMD pose on render thread
    *vr.oculus.bUpdateOnGameThred [<bool>]          Enable/Disable updating HMD pose on game thread
    *vr.Debug.VisualizeTrackingSensors [<bool>]     Enable/Disable drawing tracking sensors
    *vr.Debug.bEnableDevOverrides                   Should be queried to limit developer commands
    *vr.oculus.Debug.PositionOffset [<x> <y> <z>]   Sets position offset    
    *vr.oculus.Debug.EnforceHeadTracking [true|false|on|off|toggle] 
        Enables/Disables head tracking even if not in stereo (for testing purposes)

## Oculus Rift
    vr.oculus.bHQDistortion [on|off|toggle]         Enables/Disables high-quality distortion
    vr.oculus.PixelDensity [0.5..2.0]               Sets pixel density in the center (default is 1.0)
    vr.oculus.PixelDensity.min [0.5..2.0]           Sets minumum adaptive pixel density
    vr.oculus.PixelDensity.max [0.5..2.0]           Sets maximum adaptive pixel density
    vr.oculus.PixelDensity.adaptive [on|off|toggle] Enables/Disables adaptive pixel density
    *vr.oculus.Debug.Property bool|string|float|int <name> [value]
    *vr.oculus.Debug.bDrawGrid [on|off|toggle]
    *vr.oculus.Debug.bShowStats [on|off|toggle]
    *vr.oculus.Debug.CaptureCubemap  [xoff=?] [yoff=?] [zoff=?] [yaw=?] [gearvr] 
        All arguments are optional and default to 0. passing GEARVR makes the height of the cubemap 1024px instead of 2048

## GearVR
    vr.oculus.HandleBackButton                      Handle the 'back' button 
    vr.oculus.EnableMSAA
    vr.oculus.EnableQueueAhead                      Enable full-frame queue ahead for rendering on GearVR 
    vr.oculus.ShowGlobalMenu 
    vr.oculus.ShowQuitMenu 
    *vr.oculus.Dev.[hv]fov                          vr.Debug.bEnableDevOverides needs to be > 0 
    *vr.oculus.Debug.ToggleLoadingIconMode           

* = Not available in UE_BUILD_SHIPPING builds
